Modeling Tourist Movement Using Modified Levy Flight

The Problems

Movements of people can be measured by GPS today. However, tourist movements are complex, and it is difficult to describe the patterns only using raw GPS movement data. One of the efficient methods to understand movement patterns and processes is to use simulation models.

Objectives

This project aims to simulate tourist movements in a city using a modified Levy flight model.Â Actual tourist movement data were used to parameterize the movement model in this project. This simulation model can simplify and visualize dynamic tourist movements.
